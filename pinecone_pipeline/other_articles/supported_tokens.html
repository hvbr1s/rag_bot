This Crypto Asset List contains all the coins and tokens supported by the most recent versions of the Ledger Live app. The Crypto Asset List is available at: https://support.ledger.com/hc/articles/10479755500573-Supported-Coins-and-Tokens-in-Ledger-Live The latest version of Ledger Live for desktop or laptop computers is 2.75.0. The latest version of Ledger Live for mobile (iPhone or Android) is 3.37 How to Request a New Asset to be included in Ledger Live: If you're looking to suggest a new token or network, follow this guide to learn how to submit your request for review by our developer team at: https://developers.ledger.com/docs/tokens/integrating-tokens/evm-chains-tokens How to navigate this list To search the list, press Ctrl+F on your keyboard to activate the search tool, and then search the list by name, ticker or smart contract address. For optimal accuracy when searching for tokens, we recommend using the smart contract address as your search criteria. This list consists of both supported coins and tokens. Learn more about the difference. Token support is chain specific. For example, USDC is supported on multiple EVM chains, like Ethereum and Polygon, but not on Cardano. So, when searching a token, be sure that the token name is listed in line with the specific network you're interested in holding them on.
